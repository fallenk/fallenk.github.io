hexo使用

hexo地址

  1. 安装更新

npm install hexo-cli -g

  1. 快速开始

1.1 本地建立blog

以下命令都在blog目录下
1.2 创建md文件
$ hexo new "Hello Hexo"
1.3 生成静态网页
$ hexo generate 或者 $ hexo g
1.4 开启服务
$ hexo server 或者 $ hexo s
可进行本地查看
1.5 发布文件
$ hexo deploy或者$ hexo d
或者$ hexo g -d
发布至网上
1.6 清除缓存
$ hexo clean
