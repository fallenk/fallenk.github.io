2. 学好Java

  1. 明确自己的学习目标和大的方向，选择并锁定一门语言，按照学习方向认真研究。 目标性
  2. 初学者不用看太多的书，先找本相对基础的书进行系统的学习。 系统性
  3. 了解设计模式，编写程序，可读性，把握代码的整体结构。模式设计
  4. 不要死记语法，掌握基础语法，快速进入开发。
  5. 多实践，多思考，多请教。动手编写代码，运行程序，分析结构。
  6. 不要急躁，遇到问题，冷静思考，保持清醒的头脑，通过听歌，散步等方式。
  7. 遇到问题，首先尝试自己解决，梳理代码流程，边界情况。
  8. 多查阅资料，分析解决问题。
  9. 多阅读别人的源代码，不但要看懂别人的源代码，还要分析编程者的编程思想和设计模式。

Java语言基础

Java的主类结构

java是面向对象的程序设计语言，Java程序基本组成单元是类，类体中又包括属性和方法两部分。每一个应用程序都必须包括一个main()方法；含有main()方法的类称为主类。

包申明

一个Java应用程序是由若干类组成的。package为该类所在的包。

声明成员变量和局部变量

通常将类的属性称为全局变量(成员变量)，将方法中的属性称为局部变量。全局变量声明在类体中，局部变量声明在方法体。全局变量和变量都有各自的应用范围。

编写主方法

main()方法是类体中的主方法。该方法从”{“开始，至”}”结束。public、static和void分别是main()方法的权限修饰符静态修饰符和返回值符，Java程序中的main()方法必须声明
