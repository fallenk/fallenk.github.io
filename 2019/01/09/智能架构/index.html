因特网 vs 物联网

  • 物联网： the Internet of thing; 物物相连的互联网。
  • 技术和应用：以人为核心
  • 嵌入式 vs 物联网
    • 物联网技术并不是一个单独的技术，而是多种已有技术的融合，如互联网、嵌入式系统技术、(无线)传感网络技术等

MQTT协议分析

  1. 概述
    MQTT(Message Queuing Telemetry Transaport),消息队列遥测传输是一个client(订阅)与Server(发布)之间的消息传输协议。轻量、开源、简单的特性
    1. 传输格式精小，无应用消息头
    2. 保证消息的可靠性
      服务质量：
      1. “至多一次”: 完全依赖底层TCP/IP网络。会发生丢失或重复
      2. “至少一次”: 确保消息到达，但消息重复可能会发生
      3. “只有一次”: 确保消息到达一次
    3. MQTT在移动互联网以及移动应用中优势：
      1. 可靠传输
      2. 消息推送
      3. 低带宽
      4. 低成本
        2.1 Client
    • 连接到Server
    • 发布消息以供其他client订阅
    • 订阅其他Client的消息
    • 取消订阅
    • 与Server断开连接
      2.2 Server
    • 接收个Server的连接请求
    • 接收各Client发布的请求
    • 处理各Client的订阅/取消订阅请求
    • 推送与订阅相匹配的消息到对应Client

2.3 Application Message

- 通过MQTT协议传输的消息数据
- 当消息传输时每个消息有一个与关联的Quality of Serive 和 Topic Name
